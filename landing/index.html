OverlayKit

Design broadcast graphics once. Run them live in OBS.

A protected local Studio, editor, control panel, and browser-source runtime that move together through one open protocol.

View on GitHub Run locally

Runtime architecture

One Studio connects authoring, operation, and protected output.

01

Shows and scenes

Organize each production as a Show, then compose its scenes and reusable components.

02

Studio production

Operate variables, visibility, sounds, and scene activation from one authenticated workspace.

03

OBS browser source

Connect OBS with a rotatable, read-only output token issued by the local owner.

Quickstart

Clone, run, secure your Studio.

git clone https://github.com/OverlayKit/overlaykit.git
cd overlaykit
npm install
npm run dev:core
