OverlayKit

Design broadcast graphics once. Run them live in OBS.

A local editor, control panel, and browser-source runtime for stream overlays that move together through one protocol.

View on GitHub Run locally

Runtime architecture

Editor, panel, and OBS URL share the same motion language.

01

Visual editor

Compose scenes, lower thirds, scoreboards, tickers, alerts, and reusable collections.

02

Live panel

Operate variables, visibility, sounds, and scene activation while the overlay stays on air.

03

OBS browser source

Render the production view at /production?channel=main&transparent=true.

Quickstart

Clone, run, add the OBS URL.

git clone https://github.com/OverlayKit/overlaykit.git
cd overlaykit
npm install
npm run dev:core
