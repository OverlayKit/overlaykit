Quickstart

git clone https://github.com/OverlayKit/overlaykit.git
cd overlaykit
npm install
npm run dev:core

Open the editor on http://localhost:5174, the panel on http://localhost:5181/?channel=main, and OBS on http://localhost:5173/production?channel=main&transparent=true.
